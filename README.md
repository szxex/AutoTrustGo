## 🇯🇵 日本語 README

# AutoTrustGO

**SDXL / LoRA / LyCORIS 用・信頼度駆動型オプティマイザ**

AutoTrustGO は、
**LoRA 学習における「過学習・不安定化・AI臭さ」を抑えつつ、
細部表現と画風再現性を最大化すること**を目的として設計された
カスタム Optimizer です。

---

## 特徴

* ✅ **step 非依存設計**

  * `step_count` や warmup / decay に一切依存しません
* ✅ **confidence 駆動学習**

  * 勾配の「信頼度」を rank / stability の2成分で評価
* ✅ **過学習の自然抑制**

  * confidence 高止まり時に自動減衰
* ✅ **細部表現の保持**

  * 目・線・微細な揺れを潰しにくい
* ✅ **画風・キャラ・概念・実写に汎用対応**
* ✅ **LoRA 適用時の汎化性能が高い**
* ✅ **AdamW / Lion で起きがちな平坦化を回避**

---

## 設計思想（要点）

AutoTrustGO は
「**どれくらい信じて更新していいか**」を学習中に自己判断します。

### 重要な内部要素

* **confidence**

  * rank 成分：更新すべき構造か？
  * stability 成分：揺らしても壊れないか？
* **danger**

  * 勾配が暴走する兆候の連続評価
* **mask_ema**

  * 学習が安定している割合の記憶
* **trust_level**

  * ユーザが指定する「攻め具合」

👉 これらを **if に頼らず連続的に制御**します。

## パラメータ説明

### learning_rate

* 基本学習率
* AutoTrustGO 内部で自動スケーリングされるため、
  **AdamW よりやや高めが適正**

### trust_level（唯一の性格パラメータ）

| 値   | 挙動                   |
| --- | -------------------- |
| 0.3 | 非常に慎重                |
| 0.4 | **最推奨（再現性と汎化のバランス）** |
| 0.5 | 攻め気味（キャラ強調向け）        |

## 注意点

* ❌ step / epoch 依存の scheduler を使わないでください
* ❌ lr を極端に下げないでください
* ❌ Adam 系前提のノウハウをそのまま適用しないでください

---

このoptimizerはAdamやLionなどの既存のoptimizeと違い教師画像の特徴に徐々に進んでいくoptimizerです。（コンセプトが違います）
学習序盤で一気に進むことがなく最終的な品質は高くても途中での学習効果がわかりずらいです。
ほかのoptimizerのように短いstepでも（いいか悪いかは別として）取り敢えず何か変化するわけではないので注意してください。
step数が少ないと特徴がつかみ取る前に終了してしまうので少し多めのstep数の方が安定すると思います。
（過学習抑制はついています）

このoptimizerはAIで作成しています。AI任せではなく思想や方向性、修正内容の確認は逐次チェックしています。
そのため、著作権が発生しているかはかなりグレーですが、基本的に自由に改造、派生をしても構いません。よくなるのであれば大歓迎です。
（丸パクリしたり、コンセプトだけ取り入れて独自発想と言ったりなどの問題になるようなことをしなければ）
